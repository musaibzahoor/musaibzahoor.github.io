## Getting Started

You’ll need Node.js (recommended LTS) and Git installed.

```bash
# clone the repo (or work in your fork)
git clone https://github.com/musaibzahoor/musaibzahoor.github.io.git
cd musaibzahoor.github.io

# install dependencies
npm install

# run dev server
npm start

# build for production
npm run build
